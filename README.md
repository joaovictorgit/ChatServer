# ChatServer

## Sobre
Aplicação, desenvolvida em HTML, CSS e JS, que utiliza websockets para enviar mensagens em tempo real aos usuários que encontram-se no mesmo ambiente.

## Tecnologias

 - HTML
 - CSS
 - JavaScript
 - Socket.io
 - NodeJs
